    Creazione di un Nuovo Plugin

    Segui questa guida per creare un nuovo plugin utilizzando lo script automatico

    Prerequisiti
    • Accesso alla directory del progetto
    • Permessi di esecuzione per lo script create_plugin.sh
    Creazione del Plugin
    1. Esegui lo script di creazione:
    ./scripts/create_plugin.sh "Il Mio Plugin"
    Inserisci il nome completo del plugin tra virgolette. Lo script genererà automaticamente i nomi tecnici necessari.
    2. Struttura generata:
    il_mio_plugin/              # Nome directory (generato automaticamente)
├── __init__.py             # Configurazione del plugin
├── routes.py              # Definizione delle route
├── models.py              # Modelli del database
├── static/
│   └── il_mio_plugin/
│       ├── css/          # Fogli di stile
│       └── js/           # Script JavaScript
└── templates/
    └── il_mio_plugin/
        └── dashboard.html # Template base
    Personalizzazione

    Il plugin generato include:

    • Blueprint Flask configurato con nome visualizzato personalizzato
    • Integrazione automatica nel menu di navigazione
    • Template responsive con breadcrumb e layout standard
    • Struttura CSS/JS predisposta con stili di base
    Suggerimenti
    Nome del Plugin
    Usa un nome descrittivo e completo (es: "Gestione Utenti Avanzata")
    Riavvio Applicazione
    Riavvia Flask dopo la creazione per attivare il plugin
    Permessi Script
    Esegui chmod +x scripts/create_plugin.sh prima dell'uso
    Verifica Installazione
    Controlla i log Flask per confermare l'attivazione
